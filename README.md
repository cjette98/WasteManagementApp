# WasteManagementApp

## Run Locally

Clone the project

```bash
  git clone https://github.com/cjette98/WasteManagementApp.git
```

Go to the project directory

```bash
  cd WasteManagementApp
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  expo start -c
```
